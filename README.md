# Python Programming Immersive

## Using Python

Check that you have Python installed:

```bash
# open a new terminal

python --version

# OR

python3 --version
```

If you receive a Python version number for example `Python 3.8.13` then congratulations, you're good to go!

If your terminal gave you an error then please search for and follow the instructions for your OS to download Python onto your computer.

## Running Python

Using the Python REPL:

```bash
# open a new terminal

python

# OR

python3
```

Running Python files using the terminal:

```bash
# open a new terminal

python filename.py

# OR

python3 filename.py
```

Running a python file and then going into a REPL:

```bash
# open a new terminal

python -i filename.py

# OR

python3 -i filename.py
```
